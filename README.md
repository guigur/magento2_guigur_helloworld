# magento2_helloworld

## Installation
Add the VCS in your composer.json
        "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/guigur/magento2_helloworld"
        }],

Then execute:   
         composer require Guigur/Helloworld

## Usage
go to http://urlofmystore/helloworld
